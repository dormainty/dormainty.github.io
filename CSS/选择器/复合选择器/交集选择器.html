交集选择器 ​

作用

选中页面中同时符合多个条件的元素，交集有 并且 的含义。

语法

css
选择器1选择器2 {
	属性: 值;
}
1
2
3

注意

  • 交集选择器中如果有元素选择器，元素选择器必须写在最前面。
  • id选择器理论上可以作为交集的条件，但是实际应用中几乎不用，因为无意义。
  • 交集选择器中不可能出现两个元素选择器，因为一个元素不可能即是p元素又是span元素。
  • 用的最多的交集选择器是：元素选择器配合类选择器，例如：p.one。
css
/* 选中类名为one的p元素*/
p.one {
  color: orange;
}
/* 选中类名包含one和two的元素*/
.one.two {
  color: orange;
}
