font-weight 字体粗细 ​

语法

属性名：font-weight

常用值：

  • normal ：正常（默认值），等同于数值 400~500。
  • lighter ：细，等同于数值 100~300。
  • bold ：粗体，等同于数值 600及以上。
  • bolder ：很粗（多数字体不支持）
css
.code-better {
  font-weight: 700;
}
